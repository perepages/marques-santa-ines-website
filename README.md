# 🍷 Marqués de Santa Inès - Corporate Website

![Project Status: Pendent to finish] // some errors happened that need to bre solved + finish some features

A fully responsive, multi-page corporate website built for a wine and cava business. Designed to showcase products, present company information, and facilitate customer contact with a clean, elegant aesthetic.

Objectives:
## ✨ Features
* **Responsive Design:** Adapts perfectly to desktop, tablet, and mobile screens using media queries.
* **Custom CSS Animations:** Interactive image sliders and smooth hover transitions built entirely with CSS keyframes.
* **Dynamic Product Display:** Flexbox-based layouts for organized and visually appealing product galleries.
* **Interactive Contact Form:** Features client-side JavaScript validation to ensure accurate user input.

## 🛠️ Tech Stack
* **HTML5** (Semantic structure)
* **CSS3** (Styling, Flexbox, Keyframe Animations)
* **Vanilla JavaScript** (DOM manipulation, form validation, menu toggling)

## 💻 How to Run Locally
